# Public README Dependency Cleanup

Removed credentialed clinical corpus examples from the README so the documented
path now describes PubMed, Europe PMC, PMC Open Access, and other permitted
public/shareable sources.

Also moved the default search server, reproducibility manifest, open-image
target selection, and launcher script away from credentialed local artifact
paths. Credentialed ingest helpers remain available as explicit local-only
tooling, but they are not part of the public README or default launch path.
