# Stronger Semantic Group Borders

## Change

Increased the border weight and contrast on semantic group result sections in the search UI. The group header divider was also strengthened, and related-result group borders now use a darker red when those cards appear.

## Improvement

Semantic group boundaries are easier to see when scanning dense result grids, especially after related items are merged into the same semantic group section.

## Verification

This is a CSS-only change in `docs/search_quality/server.css`. It does not alter search behavior or result ranking.

## Remaining Limitations

No browser screenshot check was run for this small style adjustment.
